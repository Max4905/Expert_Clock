# Expert_Clock —— 一款极其简洁的教室时钟软件

<img width="466" height="218" alt="45d1ef34-3e92-4775-bffe-8e32f5584f0c" src="https://github.com/user-attachments/assets/160ff2da-0e22-49e9-b2d7-4f16d6f74692"  data-align="center" style="zoom:67%;">

本软件可以在教室大屏幕上以大字号显示时间，同时可以在不更改系统时间的情况下校正时间，以匹配学校铃声。

使用 Python Tkinter 构建，无任何联网组件，保证简洁。项目免安装，解压以后即可运行。

请注意：本软件主要功能为全屏时钟，没有在现在的、也不会在将来的版本中添加更高级的桌面挂件功能。如果需要相关功能，请移步 ClassIsland。

## 为什么是这个

- 软件足够简洁，无需安装，也不联网进行更新或者其他操作。

- 软件内显示的时间可以在不改变系统时间的情况下校准，以匹配学校铃声。

- 可以在时钟底部添加任意文本，并保留历史记录。

- 时间、日期、文本的字号均可以调节

- 具有浅色、深色等多种颜色模式。

- 可以分别切换 12/24小时制 和 读秒开关

- 所有设置保存在本地配置文件，包含一些没有对应设置界面的选项（前景、背景色以及字体名称）

## 使用方法

**注意：除非手动保存配置，否则只有程序正常退出，设置才能被保存。**

### onefile 和 onedir 的区别

* onefile 版本点击即可运行，但是启动速度稍慢；

* onedir 版本需要解压才能运行，启动更快。

### 首次运行

1. 从 [发布页面](www.github.com/Max4905/ClockForClassroom/releases) 下载最新版本

2. 运行前的准备工作
   
   - 如果下载了onefile 版本，请将文件放到一个独立文件夹里。
   - 如果下载了 onedir 版本，请将压缩包里的clock文件夹解压出来。推荐放到硬盘根目录，但这不是必须的。

3. 运行 clock.exe 。

4. （非必要）按 Ctrl-C 复制clock.exe，然后在一个方便的位置“粘贴快捷方式”。

### 校准时间

1. 从“时钟”菜单打开“校准时间”窗口。
<img width="337" height="441" alt="3d3afab5-0ef9-478f-887a-eea686700318" src="https://github.com/user-attachments/assets/6b129f60-8a4c-4d36-9df2-e833224b135e" />

2. 点击窗口底部的按钮，将时间向前调或者向后调（勾选“减小校正值”），直到第三个（加粗显示）的时间与实际时间一致。

3. 按确定保存设置。

### 更改文本

在文本框末尾添加 `--no-changer-later` 可以禁用更改文本的菜单项，直到软件再次启动。
